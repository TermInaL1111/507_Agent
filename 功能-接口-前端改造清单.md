# 功能-接口-前端改造清单（基于 fw.md）

> 来源文档：`fw.md`
> 
> 说明：
> - 本清单按“功能名 / 后端接口（URL + 方法） / 要改页面 / 期望交互”整理。
> - **接口状态分为两类**：

## 14）优先级建议（按 MVP）

### P0（立即）
1. 统一智能体交互层（AIChat + 会话）
2. 校园知识问答（含来源展示）
3. 用户认证与个人中心
4. 路由/菜单扩展

### P1（第一阶段）
1. 教师信息查询
2. 基础课表查询
3. 简单课程规划建议
4. 基础校园导航
5. 文书辅助（请假条优先）

### P2（第二阶段）
1. 选课多策略推荐
2. 校园帖子辅助
3. 地图可视化与更深层系统打通
